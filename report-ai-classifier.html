AI Image Classifier

Project report: model design, training pipeline, evaluation

Overview

Deep learning image classifier that categorizes images into target classes using CNN-based architecture.

Objectives

  • Achieve >90% accuracy on validation set.
  • Provide simple web UI for inference.
  • Enable reproducible training pipeline.

Architecture & Stack

Python TensorFlow/Keras OpenCV NumPy/Pandas Flask (UI)

CNN backbone with augmentation; optional transfer learning from pretrained models.

Training Pipeline

  • Data cleaning, label balancing, train/val split.
  • Augmentations: flip, rotate, shift, brightness.
  • Optimizer: Adam; Loss: categorical cross-entropy.
  • Callbacks: early stopping, LR scheduler, checkpoints.

Evaluation

  • Metrics: Accuracy, Precision/Recall, F1.
  • Confusion matrix for per-class insights.
  • Grad-CAM (planned) for interpretability.

Deployment

Lightweight Flask UI for uploading an image and getting predictions; can be containerized.

Future Work

  • Quantization for edge devices.
  • Better augmentation with AutoAugment.
  • On-device inference (TensorFlow Lite).
